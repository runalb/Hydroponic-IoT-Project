**Connect sensor to ESP32 GPIO Pins. GPIO pin numbers defined in the code to connect sensors respectively-**

* D2 – DHT11 Temp Sensor
* D4 – DS18B20 Water Temp Sensor
* D13 – LDR input
* D12 - ph sensor input
